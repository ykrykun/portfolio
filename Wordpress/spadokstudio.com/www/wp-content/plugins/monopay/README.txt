== Description ==

Офіційний модуль від monobank для підключення інтернет-еквайрингу.
Модуль WooCommerce для wordpress.
Тариф: 1.2% на картки Українських банків, та 1.8% на картки міжнародних банків.

Кроки для підключення:
1. Заповніть заявку за посиланням [https://docs.google.com/forms/d/1blNn0dFYPmNBR4JRDUNVzCyVmNxPQvqFZA2V3x8bO6A/viewform?edit_requested=true](https://docs.google.com/forms/d/1blNn0dFYPmNBR4JRDUNVzCyVmNxPQvqFZA2V3x8bO6A/viewform?edit_requested=true)
2. Отримайте підтвердження про відкриття терміналу (у додаток)
3. Завантажте модуль. Інструкція з установки модуля доступна за посиланням [https://doc.clickup.com/d/4e3cn-16048/monobank-woo-commercepaymentgateway](https://doc.clickup.com/d/4e3cn-16048/monobank-woo-commercepaymentgateway)

== Installation ==

= Using The WordPress Dashboard =

1. Navigate to "Plugins->Add New" from your dashboard
2. Search for 'MonoPay Woocommerce'
3. Click 'Install Now'
4. Activate the plugin

= Uploading via WordPress Dashboard =
1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings->Plugin Name screen to configure the plugin
4. Enable/disable Live mode, depending on the integration.
5. Enter your API key. MonoPay will provide a set API Key of the environments.
