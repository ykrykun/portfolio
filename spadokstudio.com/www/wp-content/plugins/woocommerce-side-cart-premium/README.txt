== Description ==

WooCommerce Side Cart adds a site wide basket icon that displays the number of products in the cart.
Customer can see all the cart items directly from the side cart & also update the quantity / remove item.
Plugin has ajax add to cart feature on single product pages too.



== Installation ==

This section describes how to install the plugin and get it working.

e.g.

= Uploading in WordPress Dashboard =

1. Navigate to the 'Add New' in the plugins dashboard
2. Navigate to the 'Upload' area
3. Select `woocommerce-side-cart.zip` from your computer
4. Click 'Install Now'
5. Activate the plugin in the Plugin dashboard

= Using FTP =

1. Download `woocommerce-side-cart.zip`
2. Extract the `woocommerce-side-cart` directory to your computer
3. Upload the `woocommerce-side-cart` directory to the `/wp-content/plugins/` directory
4. Activate the plugin in the Plugin dashboard
